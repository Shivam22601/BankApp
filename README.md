A **Bank Management System** developed using **Java** that simulates basic banking operations such as account creation, deposit, withdrawal, and balance checking.

This project demonstrates **Object-Oriented Programming (OOP)** concepts and basic **banking system functionality** implemented through a console-based application.

---

# 📌 Project Overview

The **Bank Management System** is a Java-based application designed to perform fundamental banking operations.
It allows users to manage bank accounts and perform transactions in a simple and efficient way.

This project focuses on applying **core Java programming concepts**, including classes, objects, encapsulation, and basic data management.
